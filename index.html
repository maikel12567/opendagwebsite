Software Developer

About the Program

Software development is like creating digital games and apps on a computer. Those who do this are like computer wizards who instruct the computer on what to do to make cool things happen, such as creating games or drawing programs. It's like magic for computers, and you can create amazing things with it!

Subjects/Schedule

During the program, you will study the following subjects: Dutch, English, Mathematics, and Citizenship. Additionally, you will also have the following subjects: Web Development, Native, Practice/Delta, and Practical Support.

Web Development

Here, you will learn the fundamentals of programming and how to create a website.

You will use languages such as HTML, CSS, JavaScript, PHP, and SQL in this subject.

Native

Here, you will learn how to create apps for Android and iOS.

You will use languages like C# and Python in this subject.

Practice/Delta

Here, you will learn how to approach a project and how to collaborate effectively.

Practical Support

Anything that is not programming but is related to the field will be covered in this subject.

You will also learn how to approach a project and collaborate effectively here.

Generic Subjects

Dutch, English, and Mathematics. You will learn language and story-based mathematics.

Program Structure/Blocks

Blocks: There are 2 blocks each year, and each block has its own theme. In each block, you will learn a bit about a topic. If you successfully complete a block, you move on to the next one; otherwise, you can retake the block, not the whole year.


Program Structure: There are 2 different program structures. The standard 4-year program is for everyone. You can also complete your program in 3 years if you demonstrate enough knowledge and pick it up quickly in the first half year.
